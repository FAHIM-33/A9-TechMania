###### 1. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let greeting;
greetign = {};
console.log(greetign);
```

- A: `{}`
- B: `ReferenceError: greetign is not defined`
- C: `undefined`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: A:`{}`

<i>At first a variable 'greeting' is called but the value isn't assigned. Then the value was set to and empty object. so when the value is shown in console, the output is an empty object {}</i>

</p>
</details>

###### 2. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sum(a, b) {

}

sum(1, "2");
```

- B: `TypeError`

- D: `3`

<i>If you add a number to a string, the number will be converted to a string and joined with the string. So, number 1 addition with string 2 results in string 12.</i>

###### 5. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let count = 0;
const nums = [0, 1, 2, 3];

nums.forEach((num) => {
  if (num) count += 1;
});

console.log(count);
```

- A: 1
- B: 2
- C: 3
- D: 4

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C: 3

<i>In the forEach loop, if condition is true, if num is a truthy value. In nums array, 0 is a falsy value. So count variable doesn't get incremented when num is 0. But in other num values, count is incremented. So count is 3.</i>

</p>
</details>
